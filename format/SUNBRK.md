# SUNBRK 日除けリスト

```
SUBRK
        <Sbname>
        type=<type>
        window=H<hhh>xW<www>
        D=<D>
        T=<T>
        B=<B>
        L=<L>
        R=<R>
    ;
        <Sbname>
        ...
    ;
    以下繰り返し
*
```
- Sbname: 庇の名称
- type: 以下のいずれか
   - `H`: 一般の庇
   - `HL`: 長い庇
   - `S`: 袖壁
   - `SL`: 長い袖壁
   - `K`: 格子ルーバー
   - ただし、先頭に `-` を付けると、日影部分と日商部分が逆転する。
- hhh: 高さ [m]
- www: 幅 [m]
- D: 対象面から日よけの先端までの寸法
- T: 対象領域上部の寸法
- B: 対象領域下部の寸法
- L: 左の日よけの距離
- R: 右の日よけの距離
NOTE: DTBLRの距離の単位は任意である。
