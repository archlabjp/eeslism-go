# EQPCAT（機器仕様データカタログ）

## 概要
EQPCAT（Equipment Catalog）は、建物の設備システムで使用される各種機器の性能仕様を定義するデータセットです。ボイラー、チラー、ポンプ、ファン、熱交換器、太陽光発電、蓄熱槽など、多様な設備機器の詳細な性能特性を標準化されたカタログ形式で管理できます。これにより、実際の機器性能に基づく正確なエネルギーシミュレーションが可能になります。

## データ形式
```
EQPCAT
    <機器種別> <カタログ名> <パラメータ1>=<値1> <パラメータ2>=<値2> ... [オプション] ;
    <機器種別> <カタログ名> <パラメータ1>=<値1> <パラメータ2>=<値2> ... [オプション] ;
    ... 繰り返し
*
```

## パラメータ説明

| パラメータ | 説明 | 設定値 | 参照 |
|:---|:---|:---|:---|
| 機器種別 | 機器の分類コード | BOI, REFA, COL, STANK等 | 必須 |
| カタログ名 | 機器仕様の識別名 | 任意の文字列 | 必須 |
| パラメータ | 機器種別固有の性能値 | 機器種別により異なる | 必須 |
| オプション | 追加設定・フラグ | 機器種別により異なる | 任意 |

## 使用例

### 例1: 中規模オフィスビルの空調システム
**建物概要**: 延床面積3,000m²、5階建てオフィスビル、在室者200名
**システム構成**: 中央熱源方式、VAV空調システム

```
EQPCAT
    !  熱源機器：冷暖房用ヒートポンプチラー
    REFA AirHP_150kW 
        Qo=150000        !  冷房能力150kW（ピーク負荷120kWに余裕を見込み）
        Go=7.2           !  冷水流量7.2kg/s（温度差5℃設定）
        Two=7.0          !  冷水出口温度7℃（標準的な冷水温度）
        eo=3.2           !  冷房COP3.2（高効率機種）
        Qex=140000       !  暖房能力140kW（冷房能力より若干小さい）
        Gex=6.7          !  温水流量6.7kg/s（温度差10℃設定）
        eex=3.8          !  暖房COP3.8（外気温7℃時）
        Tex=45.0         !  温水出口温度45℃（低温暖房）
        Ph=2500 ;        !  補機動力2.5kW（冷却水ポンプ等）
    
    !  搬送機器：冷温水循環ポンプ
    PUMP MainCircPump 
        type=V           !  変流量制御（省エネ運転）
        Go=7.5           !  定格流量7.5kg/s（余裕を考慮）
        Wo=11000         !  定格電力11kW（揚程40m想定）
        qef=0.85         !  発熱比率85%（モーター効率考慮）
        a0=0.1 a1=0.3 a2=0.6 ; !  特性曲線係数（実機データ）
    
    !  空調機器：給気ファン
    FAN SupplyFan 
        type=Vr          !  インバータ制御（高効率）
        Go=12.0          !  定格風量12kg/s（外気導入含む）
        Wo=22000         !  定格電力22kW（静圧1200Pa）
        qef=0.9 ;        !  発熱比率90%（ダクト内放熱）
*
```

**設計のポイント**:
- **容量設計**: ピーク負荷に対して20-25%の余裕を設定
- **効率重視**: 高効率機器の選定でランニングコスト削減
- **制御方式**: 変流量・変風量制御で部分負荷時の省エネを実現
- **温度設定**: 低温暖房（45℃）で熱源効率を向上

### 例2: 住宅用太陽光発電・蓄熱システム
**建物概要**: 戸建住宅、延床面積150m²、4人家族
**システム構成**: 太陽光発電5kW、電気蓄熱暖房、太陽熱給湯

```
EQPCAT
    !  太陽光発電：屋根設置型5kWシステム
    PV ResidentialPV_5kW 
        PVcap=5000       !  システム容量5kW（屋根面積30m²）
        Area=30.0        !  アレイ面積30m²（変換効率16.7%）
        EffInv=0.96      !  インバータ効率96%（高効率機種）
        KHD=1.0          !  日射量補正係数（標準値）
        KPD=0.97         !  経時変化補正係数（高品質パネル）
        InstallType=B    !  屋根置き型（通風あり）
        Apmax=-0.004 ;   !  温度係数-0.4%/℃（結晶系）
    
    !  太陽熱集熱器：給湯予熱用
    COL SolarWaterHeater 
        b0=0.78          !  集熱効率切片78%（平板型集熱器）
        b1=4.2 ;         !  集熱効率傾き4.2W/(m²K)（断熱性能）
    
    !  電気蓄熱暖房器：リビング用
    STHEAT LivingHeater 
        Q=6000           !  ヒーター容量6kW（夜間8時間蓄熱）
        KA=25            !  熱損失係数25W/K（高断熱仕様）
        eff=0.85         !  温風吹出効率85%（ファン付き）
        Hcap=180000      !  蓄熱体熱容量180MJ/K（レンガ蓄熱体）
        PCM=ParaffinPCM ;!  PCM併用で蓄熱密度向上
    
    !  蓄熱槽：太陽熱利用給湯システム
    STANK SolarTank 
        Vol=300          !  槽容量300L（4人家族の1.5日分）
        KAtop=1.5        !  上面熱損失1.5W/K（高断熱）
        KAside=8.0       !  側面熱損失8.0W/K（保温材厚100mm）
        KAbtm=1.2        !  底面熱損失1.2W/K（基礎断熱）
        gxr=0.1 ;        !  混合係数0.1（温度成層維持）
*
```

**設計のポイント**:
- **創エネ**: 太陽光発電で年間電力消費の70%をカバー
- **蓄エネ**: 夜間電力と太陽熱で効率的なエネルギー利用
- **省エネ**: 高断熱蓄熱槽で熱損失を最小化
- **快適性**: PCM併用で安定した暖房温度を実現

### 例3: 工場の省エネルギーシステム
**建物概要**: 製造工場、延床面積8,000m²、24時間稼働
**システム構成**: 排熱回収、デシカント空調、高効率照明

```
EQPCAT
    !  排熱回収熱交換器：製造設備からの排熱利用
    HEX WasteHeatRecovery 
        eff=0.85         !  熱交換効率85%（プレート式）
        KA=15000 ;       !  熱通過率×面積15kW/K（大容量）
    
    !  全熱交換器：外気処理用
    THEX TotalHeatExchanger 
        et=0.80          !  温度効率80%（高性能素子）
        eh=0.70 ;        !  エンタルピー効率70%（湿度回収）
    
    !  デシカント空調機：精密湿度制御
    DESI PrecisionDehumidifier 
        !  詳細パラメータはDESICCANT.mdを参照
        !  シリカゲル5kg、除湿能力20kg/h ;
    
    !  高効率ボイラー：プロセス用蒸気
    BOI ProcessBoiler 
        en=G             !  都市ガス燃料
        Qo=500000        !  定格能力500kW（蒸気1t/h）
        Qmin=100000      !  最小出力100kW（20%調節比）
        eff=0.92         !  定格効率92%（エコノマイザ付）
        Ph=5000          !  補機動力5kW（給水ポンプ等）
        blwQmin=ON ;     !  最小出力以下でもON継続
*
```

**設計のポイント**:
- **排熱回収**: 製造設備の排熱を空調・給湯に有効利用
- **湿度制御**: デシカント空調で製品品質に必要な精密制御
- **高効率化**: 全熱交換器で外気負荷を大幅削減
- **安定運転**: 最小出力以下でも連続運転で効率維持

### 例4: 病院の信頼性重視システム
**建物概要**: 総合病院、延床面積15,000m²、手術室・ICU含む
**システム構成**: 冗長化熱源、非常用発電、精密空調

```
EQPCAT
    !  主熱源：高効率ターボ冷凍機
    REFA MainChiller_300kW 
        Qo=300000        !  冷房能力300kW（2台設置で冗長化）
        Go=14.4          !  冷水流量14.4kg/s
        Two=7.0          !  冷水出口温度7℃
        eo=5.8           !  冷房COP5.8（ターボ冷凍機）
        Ph=3000 ;        !  補機動力3kW
    
    !  補助熱源：予備用ヒートポンプ
    REFA BackupHP_200kW 
        Qo=200000        !  冷房能力200kW（メンテナンス時対応）
        Go=9.6           !  冷水流量9.6kg/s
        Two=7.0          !  冷水出口温度7℃
        eo=3.5           !  冷房COP3.5（空気熱源）
        Ph=2000 ;        !  補機動力2kW
    
    !  精密空調用コイル：手術室用
    HCC PrecisionCoil 
        et=0.95          !  温度効率95%（高精度制御）
        KA=8000          !  熱通過率×面積8kW/K
        eh=0.85 ;        !  エンタルピー効率85%（除湿性能）
    
    !  非常用発電機：停電時バックアップ
    !  （注：発電機は通常EQPCATでは定義せず、電力系統として扱う）
*
```

**設計のポイント**:
- **冗長化**: 主熱源2台構成で単一故障時も運転継続
- **高効率**: ターボ冷凍機でCOP5.8の高効率運転
- **精密制御**: 手術室用コイルで温湿度±1℃、±5%RH制御
- **信頼性**: 予備機とバックアップ電源で24時間安定運転

## 設計上の注意事項

### 機器選定の指針
1. **容量設計**: 建物負荷に対する適切な機器容量
2. **効率特性**: 部分負荷特性を考慮した効率設定
3. **制御方式**: システム制御との整合性
4. **保守性**: メンテナンス性を考慮した機器選択

### カタログデータの精度
1. **メーカーデータ**: 実機器のカタログ値を使用
2. **運転条件**: 実際の運転条件での性能値
3. **劣化考慮**: 経年劣化を考慮した性能設定
4. **検証**: 実測データとの比較検証

## 屋根一体型機器の注意

**重要**: 屋根一体型太陽熱集熱器や太陽光発電は、EQPCATではなく建築部位として[WALL](WALL.md)で定義します。
