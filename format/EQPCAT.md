# EQPCAT（機器仕様データカタログ）

## 概要
EQPCAT（Equipment Catalog）は、建物の設備システムで使用される各種機器の性能仕様を定義するデータセットです。ボイラー、チラー、ポンプ、ファン、熱交換器、太陽光発電、蓄熱槽など、多様な設備機器の詳細な性能特性を標準化されたカタログ形式で管理できます。これにより、実際の機器性能に基づく正確なエネルギーシミュレーションが可能になります。

## データ形式
```
EQPCAT
    <機器種別> <カタログ名> <パラメータ1>=<値1> <パラメータ2>=<値2> ... [オプション] ;
    <機器種別> <カタログ名> <パラメータ1>=<値1> <パラメータ2>=<値2> ... [オプション] ;
    ... 繰り返し
*
```

## パラメータ説明

| パラメータ | 説明 | 設定値 | 参照 |
|:---|:---|:---|:---|
| 機器種別 | 機器の分類コード | BOI, REFA, COL, STANK等 | 必須 |
| カタログ名 | 機器仕様の識別名 | 任意の文字列 | 必須 |
| パラメータ | 機器種別固有の性能値 | 機器種別により異なる | 必須 |
| オプション | 追加設定・フラグ | 機器種別により異なる | 任意 |

## 使用例

### 基本的な機器カタログ定義
```
EQPCAT
    BOI GasBoiler_100kW en=G Qo=100000 Qmin=20000 eff=0.85 Ph=500 ;
    REFA AirHP_50kW Qo=50000 Go=2.5 Two=7.0 eo=3.5 Ph=1500 ;
    PUMP CircPump_5kW type=C Go=2.0 Wo=5000 qef=0.8 ;
    FAN Supply_Fan type=C Go=3.0 Wo=7500 qef=0.9 ;
*
```

### 再生可能エネルギー機器
```
EQPCAT
    COL SolarCollector b0=0.8 b1=4.0 ;
    PV SolarPanel_5kW PVcap=5000 Area=25.0 EffInv=0.95 InstallType=A ;
*
```

### 蓄熱・熱交換機器
```
EQPCAT
    STANK ThermalStorage Vol=5000 KAtop=2.0 KAside=10.0 KAbtm=2.0 ;
    HEX PlateHX eff=0.8 KA=5000 ;
    THEX TotalHX et=0.75 eh=0.65 ;
*
```

## 設計上の注意事項

### 機器選定の指針
1. **容量設計**: 建物負荷に対する適切な機器容量
2. **効率特性**: 部分負荷特性を考慮した効率設定
3. **制御方式**: システム制御との整合性
4. **保守性**: メンテナンス性を考慮した機器選択

### カタログデータの精度
1. **メーカーデータ**: 実機器のカタログ値を使用
2. **運転条件**: 実際の運転条件での性能値
3. **劣化考慮**: 経年劣化を考慮した性能設定
4. **検証**: 実測データとの比較検証

## 屋根一体型機器の注意

**重要**: 屋根一体型太陽熱集熱器や太陽光発電は、EQPCATではなく建築部位として[WALL](WALL.md)で定義します。
