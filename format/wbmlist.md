# wbmlist.efl 壁材料リスト

壁体材料として[WALL](./WALL.md)で引用する材料の**材料名**、**熱伝導率[W/mK]**、**容積比熱cρ [kJ/m3K]** のリストです。

書式:
```
name λ cρ
材料名 熱伝導率 W/mK 容積比熱 kJ/m3K
.
.
*
```
