# SUNBRK（日除けリスト）

## 概要
SUNBRK（Sun Break）は、建物の窓や開口部に設置される日除け装置（庇、袖壁、ルーバーなど）の形状と寸法を定義するデータセットです。これらの日除け装置による日射遮蔽効果を正確にシミュレーションすることで、建物の冷房負荷削減効果や室内の光環境を詳細に評価できます。

## データ形式
```
SUNBRK
    <日除け名称>
        type=<日除けタイプ>
        window=H<高さ>xW<幅>
        D=<出幅寸法>
        T=<上部寸法>
        B=<下部寸法>
        L=<左側寸法>
        R=<右側寸法>
    ;
    ... 繰り返し
*
```

## パラメータ説明

| パラメータ | 単位 | 説明 | 参照 |
|:---|:---|:---|:---|
| 日除け名称 | - | 日除け装置の識別名。窓定義で参照される | 必須 |
| 日除けタイプ | - | 日除けの種類を表すコード（下記参照） | 必須 |
| 高さ | m | 対象となる窓の高さ寸法 | 必須 |
| 幅 | m | 対象となる窓の幅寸法 | 必須 |
| 出幅寸法 | 任意単位 | 対象面から日除けの先端までの距離 | 必須 |
| 上部寸法 | 任意単位 | 対象領域上部の日除け寸法 | 必須 |
| 下部寸法 | 任意単位 | 対象領域下部の日除け寸法 | 必須 |
| 左側寸法 | 任意単位 | 左側の日除け寸法 | 必須 |
| 右側寸法 | 任意単位 | 右側の日除け寸法 | 必須 |

※ D、T、B、L、Rの寸法は相対的な比率として使用されるため、単位は任意です。

## 日除けタイプ一覧

| タイプコード | 名称 | 説明 | 適用場面 |
|:---|:---|:---|:---|
| `H` | 一般の庇 | 標準的な水平庇 | 南面窓の夏季日射遮蔽 |
| `HL` | 長い庇 | 長尺の水平庇 | 連続窓や大開口の遮蔽 |
| `S` | 袖壁 | 垂直の袖壁 | 東西面窓の朝夕日射遮蔽 |
| `SL` | 長い袖壁 | 長尺の垂直袖壁 | 大型開口の側面遮蔽 |
| `K` | 格子ルーバー | 格子状のルーバー | 均等な日射制御 |

### 反転機能
各タイプコードの先頭に `-` を付けることで、日影部分と日照部分を反転できます：
- `-H`: 反転庇
- `-S`: 反転袖壁
- `-K`: 反転格子ルーバー

## 使用例

### 例1: オフィスビルの統合日射制御システム
**建物概要**: 15階建てオフィスビル、カーテンウォール、年間冷房負荷30%削減目標
**設計目標**: 夏季日射遮蔽、冬季日射取得、昼光利用最大化

```
SUNBRK
    !  南面：季節対応水平庇システム
    SouthFacade_SeasonalShade
        type=H
        window=H2.8xW12.0                        !  連続窓（カーテンウォール）
        D=1.5 T=0.3 B=0.0 L=0.2 R=0.2 ;         !  出幅1.5m、上部延長0.3m
        !  設計意図：夏至の正午に窓上端まで日影、冬至は窓全面に日射
        !  遮蔽効果：6-9月の冷房期間中80%以上の日射遮蔽
        !  昼光利用：庇下面の反射で室内奥まで自然光導入
    
    !  東面：朝日制御垂直ルーバー
    EastFacade_MorningControl
        type=SL
        window=H2.8xW8.0                         !  東面連続窓
        D=0.8 T=0.0 B=0.0 L=1.2 R=0.0 ;         !  左側袖壁1.2m（建物南側）
        !  設計意図：朝の低角度日射を効果的に遮蔽
        !  遮蔽効果：6-10時の朝日を70%以上遮蔽
        !  眺望確保：垂直ルーバーで水平方向の視線確保
    
    !  西面：夕日制御複合システム
    WestFacade_EveningControl
        type=K
        window=H2.8xW8.0                         !  西面連続窓
        D=1.0 T=0.2 B=0.2 L=0.4 R=0.4 ;         !  格子ルーバー（上下左右均等）
        !  設計意図：夕方の低角度日射と高角度日射の両方を制御
        !  遮蔽効果：14-18時の西日を60%以上遮蔽
        !  通風確保：格子形状で自然通風を阻害しない
    
    !  北面：冬季日射取得重視
    NorthFacade_WinterGain
        type=H
        window=H2.8xW10.0                        !  北面窓（小さめ庇）
        D=0.4 T=0.1 B=0.0 L=0.1 R=0.1 ;         !  最小限の庇（雨除け程度）
        !  設計意図：冬季の貴重な北側日射を最大限取得
        !  採光重視：年間を通じて安定した北側光を活用
        !  雨水対策：最小限の庇で雨水の窓面流下防止
*
```

**設計のポイント**:
- **方位別最適化**: 南面は季節制御、東西面は時刻制御、北面は採光重視
- **冷房負荷削減**: 夏季日射遮蔽で冷房負荷30%削減を実現
- **昼光利用**: 庇による反射光で室内照明負荷を削減
- **自然通風**: 格子ルーバーで日射制御と通風確保を両立

**設計のポイント**:
- **パッシブデザイン**: 建築的手法で年間エネルギー消費50%削減

## 日射遮蔽の計算原理

### 影の投影計算
日除け装置による影は、太陽位置（方位角・高度角）と日除けの形状から幾何学的に計算されます：

1. **太陽位置の算出**: 日時・緯度・経度から太陽の方位角・高度角を計算
2. **影の投影**: 日除けの形状と太陽位置から窓面への影を投影
3. **遮蔽率の算出**: 影がかかる窓面積の割合を計算

### 庇の遮蔽効果
```
遮蔽率 = 影がかかる窓面積 / 全窓面積
```

### 袖壁の遮蔽効果
袖壁は主に朝夕の低角度日射に対して効果を発揮します。

## 寸法設定の指針

### 庇の設計指針
- **夏季**: 高角度の日射を遮蔽（出幅 D を大きく）
- **冬季**: 低角度の日射は取得（適度な出幅）
- **目安**: D/H = 0.3-0.8（Hは窓高さ）

### 袖壁の設計指針
- **朝夕日射**: 低角度の東西日射を遮蔽
- **目安**: L, R = 0.2-0.5（窓幅に対する比率）

### ルーバーの設計指針
- **均等遮蔽**: 一日を通じて安定した遮蔽効果
- **調整可能**: 季節に応じた角度調整が理想的

## 窓との関連付け

日除けは窓定義（WINDOW）で以下のように参照されます：
```
WINDOW
    <窓名称>
        ...
        sunbrk=<日除け名称>
        ...
    ;
*
```

## 設計上の注意事項

1. **方位の考慮**: 窓の方位に応じた適切な日除けタイプの選択
2. **季節変動**: 夏季の遮蔽と冬季の日射取得のバランス
3. **建築制約**: 構造的・意匠的制約との調整
4. **メンテナンス**: 清掃・保守の容易さを考慮

## 効果の評価指標

### 日射遮蔽率
```
日射遮蔽率 = (無遮蔽時日射量 - 遮蔽時日射量) / 無遮蔽時日射量
```

### 冷房負荷削減効果
日除けによる冷房負荷の削減量を定量的に評価できます。

## 出力データ

日除けに関する以下の項目が出力されます：
- 時刻別遮蔽率
- 日射遮蔽量
- 窓面日射量（遮蔽後）
- 冷房負荷削減効果
