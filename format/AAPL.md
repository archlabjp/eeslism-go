# AAPL（照明・機器仕様スケジュール）

## 概要
AAPL（Appliance and Lighting Schedule）は、建物内の照明器具および各種機器の発熱量とエネルギー消費量を時間スケジュールと組み合わせて定義するデータセットです。照明の発熱、機器の顕熱・潜熱発生、電力・ガス消費量を詳細に設定することで、内部発熱負荷とエネルギー消費の正確なシミュレーションが可能になります。

## データ形式
```
AAPL
    <室名>
        [ L=(<照明基準値>,<照明器具タイプ>,<照明スケジュール名>) ]
        [ As=(<機器顕熱対流成分>,<機器顕熱放射成分>,<機器顕熱スケジュール名>) ]
        [ Al=(<機器潜熱基準値>,<機器潜熱スケジュール名>) ]
        [ AE=(<電力基準値>,<電力スケジュール名>) ]
        [ AG=(<ガス基準値>,<ガススケジュール名>) ]
    ;
    ... 繰り返し
*
```

## パラメータ説明

| パラメータ | 単位 | 説明 | 参照 |
|:---|:---|:---|:---|
| 室名 | - | 対象となる室の名称。[ROOMデータセット](ROOM.md)で定義された室名を指定 | 必須 |
| 照明基準値 | W | 照明器具の基準消費電力・発熱量 | L設定時必須 |
| 照明器具タイプ | - | 照明器具の種類（現在は予約パラメータ、`x`を指定） | L設定時必須 |
| 照明スケジュール名 | - | 照明の点灯率を表すスケジュール名（0.0-1.0の比率） | L設定時必須 |
| 機器顕熱対流成分 | W | 機器からの顕熱のうち対流による成分の基準値 | As設定時必須 |
| 機器顕熱放射成分 | W | 機器からの顕熱のうち放射による成分の基準値 | As設定時必須 |
| 機器顕熱スケジュール名 | - | 機器顕熱の稼働率を表すスケジュール名（0.0-1.0の比率） | As設定時必須 |
| 機器潜熱基準値 | W | 機器からの潜熱発生量の基準値 | Al設定時必須 |
| 機器潜熱スケジュール名 | - | 機器潜熱の稼働率を表すスケジュール名（0.0-1.0の比率） | Al設定時必須 |
| 電力基準値 | W | 電力消費量集計用の基準値 | AE設定時必須 |
| 電力スケジュール名 | - | 電力消費の稼働率を表すスケジュール名（0.0-1.0の比率） | AE設定時必須 |
| ガス基準値 | W | ガス消費量集計用の基準値 | AG設定時必須 |
| ガススケジュール名 | - | ガス消費の稼働率を表すスケジュール名（0.0-1.0の比率） | AG設定時必須 |

※ すべてのスケジュール名は[SCHNMデータセット](SCHNM.md)で事前に定義する必要があります。

## 使用例

### 基本的な照明・機器設定
```
AAPL
    LivingRoom
        L=(200,x,LightingSchedule)
        As=(300,200,ApplianceSchedule) ;
    
    Kitchen
        L=(150,x,KitchenLighting)
        As=(500,300,CookingEquipment)
        Al=(200,CookingLatent) ;
*
```

### オフィスの詳細設定
```
AAPL
    Office
        L=(400,x,OfficeLighting)
        As=(800,400,OfficeEquipment)
        AE=(1200,ElectricitySchedule) ;
    
    MeetingRoom
        L=(300,x,MeetingLighting)
        As=(600,200,AVEquipment)
        AE=(900,MeetingElectricity) ;
*
```

### 住宅の包括的設定
```
AAPL
    LivingDiningKitchen
        L=(250,x,FamilyLighting)
        As=(400,300,HomeAppliances)
        Al=(150,CookingMoisture)
        AE=(800,HomeElectricity)
        AG=(300,CookingGas) ;
    
    Bedroom
        L=(100,x,BedroomLighting)
        As=(150,100,BedroomEquipment)
        AE=(250,BedroomElectricity) ;
    
    Bathroom
        L=(80,x,BathroomLighting)
        As=(200,150,BathroomEquipment)
        Al=(300,BathroomMoisture)
        AE=(500,BathroomElectricity) ;
*
```

## 発熱量の計算

### 照明発熱
照明からの発熱量は以下の式で計算されます：
```
Q_lighting = 照明基準値 × 照明スケジュール値
```

### 機器顕熱
機器からの顕熱は対流成分と放射成分に分けて計算されます：
```
Q_sensible_convection = 機器顕熱対流成分 × 機器顕熱スケジュール値
Q_sensible_radiation = 機器顕熱放射成分 × 機器顕熱スケジュール値
```

### 機器潜熱
機器からの潜熱（湿気発生）は以下の式で計算されます：
```
Q_latent = 機器潜熱基準値 × 機器潜熱スケジュール値
```

## エネルギー消費量の集計

### 電力消費量
```
Power_consumption = 電力基準値 × 電力スケジュール値
```

### ガス消費量
```
Gas_consumption = ガス基準値 × ガススケジュール値
```

## 機器種別の発熱特性

### 照明器具
- **白熱電球**: 発熱量 ≈ 消費電力
- **蛍光灯**: 発熱量 ≈ 消費電力 × 1.2（安定器含む）
- **LED**: 発熱量 ≈ 消費電力 × 0.8-1.0

### 一般機器
- **OA機器**: 主に対流、一部放射
- **調理機器**: 顕熱・潜熱両方発生
- **家電製品**: 機器により異なる放射・対流比

### 発熱成分の目安
- **対流成分**: 60-80%（一般的な機器）
- **放射成分**: 20-40%（一般的な機器）
- **潜熱成分**: 調理・洗濯等で発生

## スケジュール設定の注意事項

1. **稼働率**: 0.0（停止）から1.0（定格運転）の範囲で設定
2. **同時使用率**: 複数機器の同時使用を考慮した現実的な設定
3. **季節変動**: 照明時間や機器使用パターンの季節変化を考慮
4. **用途別特性**: 建物用途に応じた適切な使用パターンを設定

## エネルギー集計について

### AE・AGパラメータの特徴
- **独立性**: L、As、Alで設定した発熱量とは独立してエネルギー消費量を集計
- **用途**: 総エネルギー消費量の把握や省エネ効果の評価に使用
- **省略可能**: 発熱量のみが必要な場合は設定不要

### 出力ファイル
エネルギー消費量の集計結果は以下のファイルに出力されます：
- **日積算値**: `outfile_rq.es`
- **月積算値**: `outfile_dqr.ea`

## 出力データ

照明・機器に関する以下の項目が出力されます：
- 照明発熱量
- 機器顕熱量（対流・放射別）
- 機器潜熱量
- 電力消費量
- ガス消費量
- 各項目の積算値
