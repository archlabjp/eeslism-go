# WINDOW（窓リスト）

## 概要
WINDOW（Window）は、建物の窓やガラス開口部の熱的・光学的性能を定義するデータセットです。日射透過率、熱抵抗、表面放射率などのパラメータを設定することで、窓を通じた熱移動、日射取得、放射熱交換を正確にシミュレーションできます。建物の省エネルギー性能や室内環境の評価において重要な要素です。

## データ形式
```
WINDOW
    <窓名称>
        [ t=<日射透過率> ]
        [ B=<吸収日射取得率> ]
        R=<熱抵抗>
        [ Ei=<室内表面放射率> ]
        [ Eo=<外表面放射率> ]
        [ sunbrk=<日除け名称> ]
    ;
    ... 繰り返し
*
```

## パラメータ説明

| パラメータ | 単位 | 説明 | デフォルト値 | 参照 |
|:---|:---|:---|:---|:---|
| 窓名称 | - | 窓の識別名。壁体定義で参照される | - | 必須 |
| 日射透過率 | - | 窓を透過する日射の割合（0.0-1.0） | 0.0 | 任意 |
| 吸収日射取得率 | - | 窓で吸収され室内に放出される日射の割合 | 0.0 | 任意 |
| 熱抵抗 | m²K/W | 窓の熱貫流に対する抵抗値 | - | 必須 |
| 室内表面放射率 | - | 室内側表面の長波放射率（0.0-1.0） | 0.9 | 任意 |
| 外表面放射率 | - | 外側表面の長波放射率（0.0-1.0） | 0.9 | 任意 |
| 日除け名称 | - | 関連付ける日除け装置の名称 | - | 任意 |

## 使用例

### 例1: 高性能オフィスビルの窓システム
**建物概要**: 20階建てオフィスビル、南面カーテンウォール、省エネ基準適合
**設計目標**: 冷暖房負荷削減、昼光利用最大化、眺望確保

```
WINDOW
    !  南面カーテンウォール：昼光利用重視
    SouthCurtainWall_HighPerformance
        t=0.65 B=0.15 R=0.50                     !  日射透過率65%（昼光確保）、熱抵抗0.50
        Ei=0.05 Eo=0.9                           !  室内側Low-E（暖房時放射損失抑制）
        sunbrk=SouthHorizontalShade ;            !  水平庇で夏季日射制御
        !  設計意図：冬季日射取得と夏季遮蔽の両立
        !  昼光利用：透過率65%で十分な自然光確保
        !  断熱性：U値2.0W/(m²K)で省エネ基準クリア
    
    !  東西面窓：朝夕日射制御重視
    EastWestWindow_SolarControl
        t=0.45 B=0.25 R=0.45                     !  日射透過率45%（遮蔽重視）
        Ei=0.1 Eo=0.05                           !  外側Low-E（冷房時日射遮蔽）
        sunbrk=VerticalFin ;                     !  垂直フィンで朝夕日射制御
        !  設計意図：低角度日射の効果的な制御
        !  遮蔽性：透過率45%で過度な日射取得を防止
        !  眺望：適度な透過率で外部視認性確保
    
    !  北面窓：安定採光重視
    NorthWindow_StableDaylight
        t=0.75 B=0.10 R=0.55                     !  日射透過率75%（最大採光）
        Ei=0.05 Eo=0.9 ;                         !  室内側Low-E（断熱重視）
        !  設計意図：安定した北側光の最大活用
        !  採光性：透過率75%で均質な自然光確保
        !  断熱性：高断熱で熱損失最小化
    
    !  最上階役員室：プレミアム仕様
    ExecutiveFloor_Premium
        t=0.55 B=0.20 R=0.80                     !  高断熱ガラス（U値1.25）
        Ei=0.03 Eo=0.9                           !  高性能Low-E
        sunbrk=ExecutiveShade ;                  !  電動ブラインド内蔵
        !  設計意図：最高レベルの快適性と省エネ性
        !  快適性：高断熱で結露防止、温度ムラ解消
        !  制御性：電動制御で最適な環境調整
*
```

**設計のポイント**:
- **方位別最適化**: 南面は昼光利用、東西面は日射制御、北面は採光重視
- **Low-E配置**: 南面は室内側、東西面は外側で季節特性に対応
- **日除け連携**: 建築的日射制御と窓性能の相乗効果
- **省エネ基準**: 全面でU値2.0W/(m²K)以下を達成

## 窓の性能指標

### 熱貫流率（U値）
```
U値 [W/(m²K)] = 1 / 熱抵抗 [m²K/W]
```

### 日射熱取得率（η値）
```
η値 = 日射透過率 + 吸収日射取得率
```

### 遮蔽係数（SC値）
```
SC値 = η値 / 0.87（3mm透明ガラスの日射熱取得率）
```

### 単板ガラス
| 厚さ | 日射透過率 | 熱抵抗 | U値 | 用途 |

| 6mm | 0.85 | 0.16 | 6.3 | 一般建築 |
| 12mm | 0.82 | 0.18 | 5.6 | 厚板ガラス |

### 複層ガラス

| 3+A12+3 | 0.79 | 0.40 | 2.5 | 断熱複層 |
| Low-E | 0.60 | 0.50 | 2.0 | 高断熱 |

### 高性能ガラス
| 種類 | 日射透過率 | 熱抵抗 | U値 | 特徴 |

| 真空ガラス | 0.65 | 1.20 | 0.83 | 薄型高断熱 |
| 調光ガラス | 0.20-0.60 | 0.60 | 1.67 | 可変透過率 |

## 日射制御の考慮

### 方位別の日射制御
- **南面**: 夏季遮蔽・冬季取得のバランス
- **東西面**: 朝夕の低角度日射の制御
- **北面**: 安定した採光の確保

### 季節別の最適化
- **夏季**: 日射遮蔽を重視（低透過率）
- **冬季**: 日射取得を重視（高透過率）
- **中間期**: バランスの取れた性能

## 表面放射率の影響

### Low-Eコーティング
- **室内側Low-E**: 暖房時の放射熱損失を抑制
- **外側Low-E**: 冷房時の放射熱取得を抑制
- **放射率**: 0.03-0.15（通常ガラス：0.9）

### 放射熱交換
```
放射熱流 = 放射率 × ステファン・ボルツマン定数 × (T₁⁴ - T₂⁴)
```

## 窓の設置と関連付け

### 壁体での窓指定
```
WALL
    ExteriorWall
        ...
        window=<窓名称> <面積> <位置情報>
        ...
    ;
*
```

### 日除けとの組み合わせ
```
SUNBRK
    WindowShade type=H window=H2.0xW3.0 D=1.0 ... ;
*

WINDOW
    MainWindow t=0.75 R=0.40 sunbrk=WindowShade ;
*
```

## 計算における注意事項

### 熱橋の考慮
- **窓枠**: アルミ・樹脂・木製による熱橋効果
- **取付部**: 窓と壁体の接合部の断熱性
- **実効値**: 窓全体としての性能評価

### 結露対策
- **表面温度**: 室内側表面温度の計算
- **露点温度**: 室内湿度条件との比較
- **防止策**: 断熱性能の向上

## 省エネルギー基準

### 住宅の窓性能基準
| 地域区分 | U値基準 | η値基準 | 備考 |
|:---|:---|:---|:---|
| 1・2地域 | ≤2.33 | - | 寒冷地 |
| 3地域 | ≤2.33 | - | 寒冷地 |
| 4・5地域 | ≤4.65 | - | 温暖地 |
| 6・7地域 | ≤4.65 | ≤0.88 | 暖地 |

### 非住宅建築の基準
- **PAL***: 年間熱負荷係数
- **BPI**: 建築物省エネルギー性能表示制度

## 出力データ

窓に関する以下の項目が出力されます：
- 窓面温度（内外表面）
- 日射透過量
- 日射吸収量
- 熱貫流量
- 放射熱交換量
- 結露判定
