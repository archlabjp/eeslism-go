# AAPL（照明・機器仕様スケジュール）

## 概要
AAPL（Appliance and Lighting Schedule）は、建物内の照明器具および各種機器の発熱量とエネルギー消費量を時間スケジュールと組み合わせて定義するデータセットです。照明の発熱、機器の顕熱・潜熱発生、電力・ガス消費量を詳細に設定することで、内部発熱負荷とエネルギー消費の正確なシミュレーションが可能になります。

## データ形式
```
AAPL
    <室名>
        [ L=(<照明基準値>,<照明器具タイプ>,<照明スケジュール名>) ]
        [ As=(<機器顕熱対流成分>,<機器顕熱放射成分>,<機器顕熱スケジュール名>) ]
        [ Al=(<機器潜熱基準値>,<機器潜熱スケジュール名>) ]
        [ AE=(<電力基準値>,<電力スケジュール名>) ]
        [ AG=(<ガス基準値>,<ガススケジュール名>) ]
    ;
    ... 繰り返し
*
```

## パラメータ説明

| パラメータ | 単位 | 説明 | 参照 |
|:---|:---|:---|:---|
| 室名 | - | 対象となる室の名称。[ROOMデータセット](ROOM.md)で定義された室名を指定 | 必須 |
| 照明基準値 | W | 照明器具の基準消費電力・発熱量 | L設定時必須 |
| 照明器具タイプ | - | 照明器具の種類（現在は予約パラメータ、`x`を指定） | L設定時必須 |
| 照明スケジュール名 | - | 照明の点灯率を表すスケジュール名（0.0-1.0の比率） | L設定時必須 |
| 機器顕熱対流成分 | W | 機器からの顕熱のうち対流による成分の基準値 | As設定時必須 |
| 機器顕熱放射成分 | W | 機器からの顕熱のうち放射による成分の基準値 | As設定時必須 |
| 機器顕熱スケジュール名 | - | 機器顕熱の稼働率を表すスケジュール名（0.0-1.0の比率） | As設定時必須 |
| 機器潜熱基準値 | W | 機器からの潜熱発生量の基準値 | Al設定時必須 |
| 機器潜熱スケジュール名 | - | 機器潜熱の稼働率を表すスケジュール名（0.0-1.0の比率） | Al設定時必須 |
| 電力基準値 | W | 電力消費量集計用の基準値 | AE設定時必須 |
| 電力スケジュール名 | - | 電力消費の稼働率を表すスケジュール名（0.0-1.0の比率） | AE設定時必須 |
| ガス基準値 | W | ガス消費量集計用の基準値 | AG設定時必須 |
| ガススケジュール名 | - | ガス消費の稼働率を表すスケジュール名（0.0-1.0の比率） | AG設定時必須 |

※ すべてのスケジュール名は[SCHNMデータセット](SCHNM.md)で事前に定義する必要があります。

## 使用例

### 例1: 最新オフィスビルの省エネ照明・OA機器
**建物概要**: 20階建てスマートオフィス、従業員1,200名、LED照明・高効率OA機器
**設計目標**: 照明電力50%削減、OA機器効率化、昼光利用最大化

```
AAPL
    !  一般オフィスフロア：昼光利用型LED照明システム
    GeneralOffice_Floor5to18
        L=(8.0,x,DaylightLinked_LED)              !  LED照明8W/m²（従来の50%削減）
        As=(15.0,5.0,HighEfficiency_OA)           !  高効率PC・プリンター（対流15W/m²、放射5W/m²）
        AE=(25.0,TotalElectricity_Office) ;       !  総電力消費量25W/m²（照明+OA+その他）
        !  照明：昼光センサー連動で自動調光、在席センサーで個別制御
        !  OA機器：省エネPC、LED液晶、スリープ機能活用
        !  電力管理：リアルタイム電力監視でピークカット
    
    !  役員フロア：高品質照明・プレミアムOA環境
    ExecutiveFloor_19to20
        L=(12.0,x,Premium_LED)                    !  プレミアムLED照明12W/m²
        As=(20.0,8.0,Executive_Equipment)         !  高性能ワークステーション
        AE=(35.0,Executive_Electricity) ;         !  総電力消費量35W/m²
        !  照明：調色機能付きLED、個別調光制御
        !  OA機器：高性能PC、大型ディスプレイ、高級AV機器
    
    !  会議室：プレゼンテーション対応照明・AV設備
    ConferenceRoom_Large
        L=(15.0,x,Presentation_Lighting)          !  プレゼン用調光照明15W/m²
        As=(25.0,15.0,AV_Equipment)               !  プロジェクター・音響機器
        AE=(45.0,Meeting_Electricity) ;           !  会議室総電力45W/m²
        !  照明：調光・調色対応、プレゼン時自動減光
        !  AV機器：4Kプロジェクター、音響システム、Web会議設備
    
    !  データセンター：サーバー室の高密度機器
    DataCenter_Floor1
        L=(5.0,x,DC_Emergency_Lighting)           !  非常用照明のみ5W/m²
        As=(200.0,50.0,Server_Equipment)          !  サーバー・ネットワーク機器
        AE=(300.0,DC_TotalPower) ;                !  データセンター総電力300W/m²
        !  照明：通常は消灯、保守時のみ点灯
        !  サーバー：高密度配置、排熱250W/m²
*
```

**設計のポイント**:
- **LED化**: 従来蛍光灯比50%の電力削減（8W/m² vs 16W/m²）
- **昼光利用**: 窓際は昼光センサーで照明を自動調光
- **OA機器効率**: 省エネ認定機器で消費電力30%削減
- **電力監視**: リアルタイム監視でピーク電力制御

## 発熱量の計算

### 照明発熱
照明からの発熱量は以下の式で計算されます：
```
Q_lighting = 照明基準値 × 照明スケジュール値
```

### 機器顕熱
機器からの顕熱は対流成分と放射成分に分けて計算されます：
```
Q_sensible_convection = 機器顕熱対流成分 × 機器顕熱スケジュール値
Q_sensible_radiation = 機器顕熱放射成分 × 機器顕熱スケジュール値
```

### 機器潜熱
機器からの潜熱（湿気発生）は以下の式で計算されます：
```
Q_latent = 機器潜熱基準値 × 機器潜熱スケジュール値
```

## エネルギー消費量の集計

### 電力消費量
```
Power_consumption = 電力基準値 × 電力スケジュール値
```

### ガス消費量
```
Gas_consumption = ガス基準値 × ガススケジュール値
```

## 機器種別の発熱特性

### 照明器具
- **白熱電球**: 発熱量 ≈ 消費電力
- **蛍光灯**: 発熱量 ≈ 消費電力 × 1.2（安定器含む）
- **LED**: 発熱量 ≈ 消費電力 × 0.8-1.0

### 一般機器
- **OA機器**: 主に対流、一部放射
- **調理機器**: 顕熱・潜熱両方発生
- **家電製品**: 機器により異なる放射・対流比

### 発熱成分の目安
- **対流成分**: 60-80%（一般的な機器）
- **放射成分**: 20-40%（一般的な機器）
- **潜熱成分**: 調理・洗濯等で発生

## スケジュール設定の注意事項

1. **稼働率**: 0.0（停止）から1.0（定格運転）の範囲で設定
2. **同時使用率**: 複数機器の同時使用を考慮した現実的な設定
3. **季節変動**: 照明時間や機器使用パターンの季節変化を考慮
4. **用途別特性**: 建物用途に応じた適切な使用パターンを設定

## エネルギー集計について

### AE・AGパラメータの特徴
- **独立性**: L、As、Alで設定した発熱量とは独立してエネルギー消費量を集計
- **用途**: 総エネルギー消費量の把握や省エネ効果の評価に使用
- **省略可能**: 発熱量のみが必要な場合は設定不要

### 出力ファイル
エネルギー消費量の集計結果は以下のファイルに出力されます：
- **日積算値**: `outfile_rq.es`
- **月積算値**: `outfile_dqr.ea`

## 出力データ

照明・機器に関する以下の項目が出力されます：
- 照明発熱量
- 機器顕熱量（対流・放射別）
- 機器潜熱量
- 電力消費量
- ガス消費量
- 各項目の積算値
