# EVAC（気化冷却器）

## 概要
EVAC（Evaporative Cooler：気化冷却器）は、水の蒸発潜熱を利用して空気を冷却する機器です。直接気化冷却と間接気化冷却の両方に対応しています。

## データ形式

### 機器仕様（EQPCAT）
```
*EQPCAT
EVAC
機器名 分割数 Wet側面積 Dry側面積 Wet側熱伝達率 Dry側熱伝達率 積層数
```

### システム構成要素（SYSCMP）
```
*SYSCMP
EVAC 機器名 カタログ名
```

## パラメータ説明

| パラメータ | 単位 | 説明 |
|------------|------|------|
| 機器名 | - | 気化冷却器の識別名 |
| 分割数 | - | 計算用の分割数 |
| Wet側面積 | m² | 湿り側の境界壁接触面積 |
| Dry側面積 | m² | 乾き側の境界壁接触面積 |
| Wet側熱伝達率 | W/(m²·K) | 湿り側の対流熱伝達率 |
| Dry側熱伝達率 | W/(m²·K) | 乾き側の対流熱伝達率 |
| 積層数 | - | 熱交換部の積層数 |

## 使用例

### カタログデータ定義
```
*EQPCAT
EVAC
EvapCooler_01 10 50.0 50.0 25.0 25.0 5
```

### システムでの使用
```
*SYSCMP
EVAC EvapCooler_System_01 EvapCooler_01
```

## 計算方法

### 熱・物質移動
気化冷却器では以下の現象が同時に計算されます：
- **顕熱交換**: Dry側とWet側の温度差による熱交換
- **潜熱交換**: 水の蒸発による潜熱移動
- **物質移動**: 水蒸気の移動

### 効率計算
- **温度効率**: 乾球温度の変化効率
- **エンタルピ効率**: 全熱（エンタルピ）の変化効率

## 出力データ

気化冷却器の運転状態として以下が出力されます：
- Dry側・Wet側の入口・出口温度
- Dry側・Wet側の入口・出口湿度
- 顕熱・潜熱・全熱の処理量
- 蒸発水量

## 適用範囲

### 適用条件
- 外気温度が高く、湿度が低い地域
- 工場や倉庫などの大空間
- 省エネルギー冷房システム

### 制限事項
- 高湿度環境では効果が限定的
- 水質管理が必要
- 冬季の凍結対策が必要

## 注意事項

1. **水質**: 冷却水の水質管理が重要です
2. **メンテナンス**: 定期的な清掃・交換が必要です
3. **湿度制御**: 室内湿度の上昇に注意が必要です
4. **気候条件**: 乾燥地域での使用が効果的です
